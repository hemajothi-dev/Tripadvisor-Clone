# Tripadvisor-Clone


A clean and responsive TripAdvisor clone built using only HTML and CSS. This project replicates the visual design of the TripAdvisor homepage, featuring a navigation bar, search section, destination cards, and travel themes. It is a front-end design project meant for learning, showcasing web design skills, or using as a base for further development.
